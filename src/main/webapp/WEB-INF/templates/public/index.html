À propos de Color Run

Color Run est né d'une idée simple : rendre la course à pied amusante et accessible à tous. Notre mission est de créer des expériences uniques où le sport rencontre le divertissement.

Pour tous les niveaux

Que vous soyez un coureur aguerri ou un débutant, nos courses sont conçues pour tous les niveaux.

Une explosion de couleurs

À chaque kilomètre, vous traverserez des zones de poudre colorée, transformant votre course en une expérience visuelle unique.

Une communauté vibrante

Rejoignez des milliers de participants qui partagent votre passion pour le sport et le fun.
